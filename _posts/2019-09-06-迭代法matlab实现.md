## 代码实现

> 二分法：求f（x）=x^3+10x-20 在区间（1，2）之间的根，取误差e=10^-4

```matlab
% bipart.m
function [x,m]=bipart(fun,a0,b0,tol)
%x为近似根，m为迭代次数
a=a0;b=b0;
m=1+round(round(log((b-a)/tol))/log(2));
for k=1:m
    p=(a+b)/2;     
    if fun(p)*fun(b)<0
            a=p;
    else
            b=p;
    end
    x=p;
end
end
```

```matlab
function f=fun1(x)
% fun1.m:
f=x^3+10*x-20;
```

> 调用：[x,m]=bipart(@fun1,1,2,0.0001)
> 结果: x = 1.5945 m = 14
